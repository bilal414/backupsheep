Getting Started

1. Setup Integration

Setup integration to connect with your cloud hosting account, websites, database and storage accounts.

2. Setup Node

Using the integration, create a backup node which can be a cloud server, volume, database or files/folders.

3. Setup Schedule

You can configure schedule(s) for each node and choose when and where to store your backups.

4. Setup Storage (optional)

Bring your storage accounts for website and database backups. Cloud server snapshots are not downloadable.
