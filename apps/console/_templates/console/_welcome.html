Getting started

4 steps
  • Step 1

    Set up an integration

    Connect your cloud hosting account, websites, databases and storage accounts.

    Connect
  • Step 2

    Set up a node

    Using the integration, create a backup node — a cloud server, volume, database or files/folders.

  • Step 3

    Set up a schedule

    Configure schedule(s) for each node and choose when and where to store your backups.

  • Step 4

    Set up storage optional

    Bring your own storage accounts for website and database backups. Cloud server snapshots are not downloadable.
